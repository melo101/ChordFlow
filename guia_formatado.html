Guia Completo - ChordFlow

1. LISTA DE REPERTÓRIOS
----------------------
Este é o ecrã principal que mostra todos os seus repertórios de músicas.

• Ver Repertórios: A lista mostra todos os repertórios criados
• Adicionar Repertório: Toque no botão amarelo com "+" no canto inferior direito
• Abrir Repertório: Toque em qualquer repertório para ver as músicas que contém
• Menu de Contexto: Pressione prolongadamente num repertório para aceder a opções adicionais como eliminar ou exportar o repertório

2. LISTA DE MÚSICAS
------------------
Este ecrã mostra todas as músicas num repertório específico.

• Ver Músicas: A lista mostra todas as músicas no repertório selecionado
• Adicionar Música: Toque no botão amarelo com "+" para criar uma nova música
• Abrir Música: Toque em qualquer música para ver os seus acordes
• Pesquisar: Utilize a caixa de pesquisa para encontrar músicas rapidamente
• Alterar Ordem: Utilize os botões flutuantes para alterar a ordem das músicas no repertório
• Menu de Contexto: Pressione prolongadamente numa música para editar ou eliminar

3. ECRÃ DE DETALHE DA MÚSICA
---------------------------
Este ecrã mostra os acordes para tocar.

• Ver Música: Veja os acordes formatados por secções
• Deslocar: Deslize para cima e para baixo para ver toda a música
• Zoom: Utilize o gesto de pinça (dois dedos) para aumentar ou diminuir o tamanho do texto
• Repor Zoom: Toque duas vezes para restaurar o tamanho original do texto
• Transpor: Toque no botão de transposição (ícone de nota musical) para abrir o diálogo de transposição
• Diálogo de Transposição: Ajuste o número de semitons usando a barra deslizante e escolha entre notação com sustenidos (#) ou bemóis (b)
• Voltar à Lista: Deslize da esquerda para a direita para voltar à lista de músicas

4. ECRÃ DE EDIÇÃO DE MÚSICA
-------------------------
Este ecrã permite-lhe criar ou modificar uma música.

• Editar Título: Preencha o nome da música
• Definir Tonalidade: Especifique a tonalidade original da música (por ex., C, D, Am)
• Adicionar Secções: Digite o nome da secção (por ex., Verso, Refrão), digite os acordes da secção e toque em "Adicionar Secção"
• Editar Secções: Pressione prolongadamente numa secção já adicionada para a editar ou remover
• Guardar Música: Toque no botão "Guardar Música" na parte inferior do ecrã

5. GESTOS E INTERAÇÕES ESPECIAIS
------------------------------
• Deslizar no Ecrã de Detalhes: Deslize da esquerda para a direita para voltar à lista de músicas
• Zoom no Ecrã de Detalhes: Belisque com dois dedos para aumentar/diminuir o texto; Toque duplo para restaurar o tamanho original
• Menu de Contexto: Pressione prolongadamente num item para ver opções adicionais
