GUIA DO UTILIZADOR - CHORDFLOW

1. LISTA DE REPERTÓRIOS

Este é o ecrã principal que mostra todos os seus repertórios de músicas.

  • Ver Repertórios: A lista mostra todos os repertórios criados
  • Adicionar Repertório: Toca no botão amarelo com "+" no canto inferior direito
  • Abrir Repertório: toca em qualquer repertório para ver as músicas que contém
  • Menu de Contexto: Pressiona prolongadamente num repertório para aceder a opções adicionais como eliminar ou exportar o repertório
2. LISTA DE MÚSICAS

Este ecrã mostra todas as músicas num repertório específico.

  • Ver Músicas: A lista mostra todas as músicas no repertório selecionado
  • Adicionar Música: Toca no botão amarelo com "+" para criar uma nova música
  • Abrir Música: Toca em qualquer música para ver os seus acordes
  • Pesquisar: Utiliza a caixa de pesquisa para encontrar músicas rapidamente
  • Alterar Ordem: Utiliza os botões flutuantes para alterar a ordem das músicas no repertório
  • Menu de Contexto: Pressiona prolongadamente numa música para editar ou eliminar
3. ECRÃ DE DETALHE DA MÚSICA

Este ecrã mostra os acordes para tocar.

  • Ver Música: Vê os acordes formatados por secções
  • Deslocar: Desliza para cima e para baixo para ver toda a música
  • Zoom: Utiliza o gesto de pinça (dois dedos) para aumentar ou diminuir o tamanho do texto
  • Repor Zoom: Toca duas vezes para restaurar o tamanho original do texto
  • Transpor: Toca no botão de transposição (ícone de nota musical) para abrir o diálogo de transposição
  • Diálogo de Transposição: Ajusta o número de semitons usando a barra deslizante e escolhe entre notação com sustenidos (#) ou bemois (b)
  • Voltar à Lista: Desliza da esquerda para a direita para voltar à lista de músicas
4. ECRÃ DE EDIÇÃO DE MÚSICA

Este ecrã permite criar ou modificar uma música.

  • Editar Título: Preenche o nome da música
  • Definir Tonalidade: Especifica a tonalidade original da música (por ex., C, D, Am)
  • Adicionar Secções: Digita o nome da secção (por ex., Verso, Refrão), digita os acordes da secção e toca em "Adicionar Secção"
  • Editar Secções: Pressiona prolongadamente numa secção já adicionada para a editar ou remover
  • Guardar Música: Toca no botão "Guardar Música" na parte inferior do ecrã
5. GESTOS E INTERAÇÕES ESPECIAIS
  • Deslizar no Ecrã de Detalhes: Desliza da esquerda para a direita para voltar à lista de músicas
  • Zoom no Ecrã de Detalhes: Belisca com dois dedos para aumentar/diminuir o texto; Toca duplo para restaurar o tamanho original
  • Menu de Contexto: Pressiona prolongadamente num item para ver opções adicionais
6. TRANSPOSIÇÃO DE TOM

Recurso para mudar o tom de uma música para acomodar vocais ou instrumentos diferentes.

  1. Abre uma música no ecrã de detalhes
  2. Toca no botão de transposição (ícone de nota musical) para abrir o diálogo de transposição
  3. Ajusta o número de semitons usando a barra deslizante (valores positivos para subir o tom, negativos para baixar)
  4. Escolhe a notação preferida: sustenidos (#) ou bemois (b) nas definições ou no diálogo de transposição
  5. Toca em "Aplicar" para ver os acordes transpostos
  6. O tom atual é exibido ao lado do título da música
  7. A transposição é guardada automaticamente

Nota: A notação com bemois é particularmente útil para instrumentos como saxofone, trompete e outros instrumentos de sopro que normalmente usam essa notação.

7. MÚSICAS COM LETRAS E ACORDES

O ChordFlow permite adicionar músicas com letras e acordes embutidos, facilitando a leitura e execução durante a performance.

  1. Toca no botão "+" na lista de músicas
  2. Seleciona "Adicionar Música com Letras"
  3. Preenche o título e o tom da música
  4. Adiciona as linhas de acordes começando com um ponto (.) e posicionando os acordes diretamente acima das sílabas correspondentes na linha seguinte:

[Verso]

. C G Am F

Imagine there is no heaven

. C G Am F

It is easy if you try

  1. Para adicionar acordes isolados, coloca-os entre colchetes, por exemplo: [D]
  2. Para indicar secções, use o nome da secção entre colchetes numa linha separada, como [Verso] ou [Refrão]
  3. Toca em "Guardar" para salvar a música

As músicas com letras e acordes são identificadas pelo ícone de letras na lista de músicas e podem ser transpostas da mesma forma que as músicas apenas com acordes.

8. IMPORTAÇÃO DE MÚSICAS

O ChordFlow suporta a importação de músicas em diferentes formatos, permitindo que prepares as tuas músicas num editor de texto e importes todas de uma vez.

  1. No menu principal, seleciona "Importar Músicas"
  2. Escolha o tipo de importação:
  • Formato Compacto (Secções): Para músicas com apenas acordes organizados por secções
  • Formato Multilinha (Letras e Acordes): Para músicas com letras e acordes embutidos
  1. Cola o texto da música no formato adequado
  2. Verifica as instruções de formato no ecrã de importação para garantir que o texto está no formato correto
  3. Toca em "Importar" para adicionar a música ao teu repertório

Dica: Podes preparar várias músicas num editor de texto e importá-las em sequência.

Exemplos de Formatos:

Formato Compacto: Separe as músicas por uma quebra de linha (espaço em branco).

WONDERFUL TONIGHT;G;INTRO,G D C D G D C D|VERSE,G D C D G D C D|CHORUS,C D G Em C D G|

BOHEMIAN RHAPSODY;Bb;INTRO,Bbmaj7 Gm7 Cm7 F7|VERSE,Bbmaj7 Gm7 Cm7 F7|BRIDGE,Bbmaj7 Eb Bbmaj7 Eb|CHORUS,F7 Bbmaj7 Ebmaj7 Bbmaj7

9. DICAS E TRUQUES
  • No ecrã de detalhes, usa o zoom para ajustar o tamanho do texto conforme a sua necessidade de visualização
  • Configure o seu dispositivo para não entrar em modo de espera durante ensaios ou atuações
  • Usa o gesto de deslizar para alternar rapidamente entre músicas durante uma atuação
  • Organiza as tuas músicas em repertórios temáticos ou baseados em ocasião para encontrá-las facilmente

11. INSTALAÇÃO DA VERSÃO WINDOWS

A versão Windows do ChordFlow agora é distribuída através de um instalador que facilita o processo de instalação. Ainda assim, podes receber alguns avisos de segurança do Windows Defender ou do teu antivírus. Isto acontece porque:

  • O ChordFlow não é assinado com um certificado comercial (que é caro)
  • O Windows SmartScreen pode exibir um aviso sobre uma "aplicação não reconhecida"

Importante: O ChordFlow é completamente seguro e não contém malware ou vírus.

Para instalares o ChordFlow no Windows:

  1. Descarrega o instalador do nosso site
  2. Se vires uma mensagem de aviso do Windows SmartScreen, clica em "Mais informações"
  3. Em seguida, clica em "Executar assim mesmo"
  4. Segue as instruções do assistente de instalação para concluir o processo
  5. Após a instalação, o ChordFlow estará disponível no menu Iniciar do Windows

A aplicação não recolhe dados pessoais e funciona totalmente offline. Podes verificar a nossa Política de Privacidade para mais informações.
